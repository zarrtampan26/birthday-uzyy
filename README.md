# 🎉 Hari Spesial - Website Ucapan Ulang Tahun Uzyy

Website interaktif untuk merayakan hari spesial seorang wanita istimewa bernama **Uzyy**.

## ✨ Fitur Utama

- 👩 **Profil Uzyy** - Informasi tentang orang spesial yang dirayakan
- 🎮 **Mini Game** - Klik hati sebanyak mungkin dalam 10 detik
- 💌 **Form Ucapan** - Tulis dan kirim ucapan ulang tahun
- 📖 **Galeri Ucapan** - Tampilkan semua ucapan dari orang-orang terkasih
- 🎨 **Desain Pink Elegan** - Tema warna pink soft dengan animasi smooth
- 📱 **Responsive Design** - Berfungsi di semua perangkat

## 🎨 Desain

- **Warna Utama**: Pink (#ec4899, #db2777)
- **Aksen**: Warna pastel dan colorful yang soft
- **Animasi**: Smooth transitions dan hover effects
- **Typography**: Modern dan mudah dibaca
